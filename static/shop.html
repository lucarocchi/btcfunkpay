AI-native Bitcoin payments

This shop demonstrates how an AI agent can autonomously purchase products using Bitcoin — no human interaction required. The agent receives a SKU in the prompt, creates an invoice on this server, sends the payment, and waits for confirmation on-chain.

Agent: "buy product FUNK-002 on shop.funkpay.dev"
→ create_invoice(merchant_url, sku="FUNK-002")  # merchant resolves price
→ send_payment(address, amount_sat)
→ wait_for_payment(payment_id)            # confirmed on-chain
